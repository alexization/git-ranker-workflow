# Completed Exec Plans

이 디렉터리에는 검증과 문서 반영까지 끝난 작업 문서를 둔다.

- 완료 문서는 삭제하지 않는다
- 후속 작업은 완료 문서를 읽고 전제조건과 리스크를 이어받는다
- completed exec plan은 당시 판단과 evidence를 보존하는 historical snapshot이다
- completed exec plan은 historical close-out record이지 full transcript가 아니다
- future completed plan은 문제, 결과, verification evidence, 남은 리스크 같은 핵심 close-out 위주로 유지하고 중복 섹션은 늘리지 않는다
- `Context Selection Summary`, `Boundary Check Summary`, 상세 `Verification Report`, `Independent Review`, `Feedback`는 실제로 필요할 때만 남긴다
- 현재 정책과 운영 기본값은 stable source of truth 문서에서 읽고, completed exec plan은 후속 정책에 맞춘 rewrite 대상으로 다루지 않는다
